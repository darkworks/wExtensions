# .caretPosition()

## About

Get caret position index in textarea or text input.  Also set the caret position or highlight range.

## Examples

```js
$('#demo-textarea').caretPosition(); // 3
$('#demo-textarea').caretPostion(5);
$('#demo-textarea').caretPostion(5, 10);
```
