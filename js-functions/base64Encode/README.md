# window.base64Encode()

## About

Base64 encode a string.  Note that this also requires the [utf8Encode](https://github.com/websanova/wExtensions/tree/master/js-functions/utf8Encode) function.

## Examples

```js
window.base64Encode("testing123"); // "dGVzdGluZzEyMw=="
```
