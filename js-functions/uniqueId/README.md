# window.uniqueId()

## About

A JavaScript funciton to return a random string.  The default length is 6 but it also accetps an argument for length.

## Examples

```js
window.uniqueId(); // "a7h3ca"
window.uniqueId(10); // "8fha6r6ajc"
```
